## Built-in LLM tools

These docs ship with LLM-friendly tools so you can ask questions, generate code, and summarize pages directly from the UI.

You can use:

- **ChatGPT**
- **Claude**
- **Perplexity**
- **Cursor**
- **VS Code**
- **MCP**

## How it works

The docs UI exposes a contextual menu on each page. Use it to copy page content or send it to an LLM with one click.

## Tips

- Ask for an integration checklist based on your use case.
- Paste the webhook section and ask for a handler in your stack.
- Ask for a curl example using your real endpoint and API key.
